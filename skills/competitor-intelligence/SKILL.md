# Competitor Intelligence Skill

## 使用场景

当用户需要分析山东寿光黄瓜、河北黄瓜、辽宁某批发市场黄瓜等区域供应源竞品时触发本 Skill。输入可以是区域供应源名称、网页 URL、手工 CSV 数据或一段公开行情/监管/行业资讯文本。

## 输入

- `competitor`：区域供应源竞品名称，例如“山东寿光黄瓜”。代码中暂时保留该字段名，但业务含义不是品牌、公司或电商平台。
- `query`：分析问题，例如“分析寿光黄瓜相对河北黄瓜的批发价差和质量风险”。
- `evidence`：公开网页、CSV、RAG 检索结果或人工整理文本。

## 输出

- 一句话结论。
- 价格波动、区域价差、新产季/新品种/新规格/新批次供应、质量与监管风险等关键发现。
- 可执行建议。
- 证据来源 URL 或文件路径。

## 使用示例

```text
请使用 competitor-intelligence 分析山东寿光黄瓜近期批发价变化，并和河北黄瓜、辽宁批发市场黄瓜做对比。
```

## 执行规则

1. 优先使用公开、可追溯的数据源，不绕过登录、验证码或反爬限制。
2. 结论必须基于证据；证据不足时明确说明缺口。
3. 报告面向运营决策，避免空泛评价。
