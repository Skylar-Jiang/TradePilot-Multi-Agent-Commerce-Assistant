# 数据源说明

默认数据源配置位于：

```text
config/sources.yaml
```

数据源分为四类：

1. 政府/行业价格数据：全国农产品批发市场价格信息、商务部食用农产品价格信息等。
2. 区域/批发市场行情：山东寿光黄瓜、河北黄瓜、辽宁批发市场黄瓜等供应源的批发价、到货价、单位价和区域价差，可通过手工 CSV 或授权 API 接入。
3. 供应动态与行业资讯：新产季上市、新品种、新产区、新规格、新批次供应、到货量变化、天气物流影响等。
4. 质量监管与供应风险：抽检不合格、食品安全、农残/兽残、腐损率、缺斤少两、供应短缺、监管通报等公开信息或手工样例。

系统不绕过登录、验证码、反爬，也不做 App 抓包。真实业务数据必须通过公开网页、授权 API 或手工 CSV 接入。

CSV 字段统一为：

```text
id, competitor, product_category, product_name, specification, price, unit_price, title, content, source, publish_time, url, dimension
```

其中 `competitor` 在本项目中指“区域供应源竞品”，例如“山东寿光黄瓜”“河北黄瓜”“辽宁批发市场黄瓜”。
