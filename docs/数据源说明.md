# 数据源说明

默认数据源配置位于：

```text
config/sources.yaml
```

数据源分为四类：

1. 政府/行业价格数据：全国农产品批发市场价格信息、商务部食用农产品价格信息等。
2. 平台价格与促销：京东生鲜、盒马、叮咚买菜、美团买菜等手工 CSV 或授权 API。
3. 行业媒体：36氪、界面新闻、新京报、证券时报、钛媒体、中国商报、食品行业媒体等，可通过公开页或手工 CSV 接入。
4. 社交与投诉舆情：黑猫投诉、知乎、小红书、微博、App Store/应用商店评论等公开样例手工 CSV。

系统不绕过登录、验证码、反爬，也不做 App 抓包。平台 App 数据必须通过授权 API 或手工 CSV 接入。

CSV 字段统一为：

```text
id, competitor, product_category, product_name, specification, price, unit_price, title, content, source, publish_time, url, dimension
```
