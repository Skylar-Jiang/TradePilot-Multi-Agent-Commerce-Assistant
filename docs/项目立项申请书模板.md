**一、 项目背景与目标**

- **背景**: 生鲜批发采购需要持续关注不同区域、产区、批发市场和供应渠道的价格波动、到货稳定性、质量风险和监管公告。实时感知区域供应源动态，有助于采购商控制成本、规避质量风险并优化备货节奏。
- **目标**: 构建一个自动化区域供应源竞品情报分析系统，能够从公开行情、监管公告、行业资讯和手工 CSV 中提取供应源动态，输出结构化的采购对标简报。

**二、 监控维度与竞品列表**

- **监控维度**:
  1. **价格战**: 监控区域供应源之间的批发价、到货价、市场价、异常价差和采购优势。
  2. **新品发布**: 在本项目中理解为新产季上市、新品种、新产区、新规格、新批次供应。
  3. **负面舆情**: 在本项目中理解为食品安全、抽检不合格、质量风险、供应短缺、天气物流影响和监管通报。

**三、 技术选型与架构**

- **大模型**: OpenAI-compatible Provider / DeepSeek 兼容网关
- **开发框架**: LangChain (Chains, Agents, RAG, Memory)
- **后端服务**: FastAPI
- **向量数据库**: Chroma
- **开发语言**: Python 3.11+

**四、 开发计划**
