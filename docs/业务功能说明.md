# 生鲜电商竞品动态追踪与价格舆情分析业务功能说明

本系统面向盒马、叮咚买菜、美团买菜、朴朴超市、京东七鲜、永辉生活、山姆会员店、京东生鲜等平台，追踪肉蛋奶、基础蔬菜、水果、水产、预制菜等生鲜商品的价格波动、促销变化、新品/节令上架、负面舆情和平台对比。

选择生鲜电商作为测试对象，是因为生鲜价格每天变化明显，肉蛋奶、蔬菜、水果属于高频消费品，数据贴近日常生活，适合验证价格监控、时序分析、舆情识别和报告生成。

核心功能：

1. 多源数据采集：支持政府/行业价格公开页、授权 API、行业媒体、手工 CSV。
2. 价格监控：识别鸡蛋、牛奶、猪肉、番茄、黄瓜、苹果、草莓等商品价格和单位价。
3. 新品与节令商品分析：识别草莓季、车厘子预售、年货礼盒、预制菜新品。
4. 舆情风险分析：识别不新鲜、缺斤少两、配送慢、包装破损、售后差、价格虚高。
5. 趋势对比：按日/周/月沉淀价格和舆情数据，支持平台对比。
6. 报告生成：输出生鲜电商竞品分析报告。

系统不绕过登录、验证码、反爬，不抓取隐私或付费内容。知乎、小红书、微博、App 评论等数据通过公开样例手工 CSV 导入。
