# 生鲜批发采购区域供应源竞品动态追踪与智能对标分析业务功能说明

本系统面向生鲜批发商、采购商和供应链人员，围绕黄瓜、番茄、鸡蛋、猪肉等具体生鲜品类，追踪不同区域、产区、批发市场和供应渠道提供的同类商品动态。例如：山东寿光黄瓜、河北黄瓜、辽宁批发市场黄瓜。

在本项目中，代码和接口中的 `competitor` 字段暂时保留，业务含义统一为“区域供应源竞品”，不是品牌、公司或电商平台。

选择生鲜批发采购作为测试对象，是因为生鲜批发价、到货价和区域价差变化频繁，且直接影响采购成本、备货节奏、供应稳定性和质量风险判断，适合验证价格监控、RAG 证据检索、多 Agent 分析和报告生成。

核心功能：

1. 多源数据采集：支持政府/行业价格公开页、批发市场行情、监管公告、天气物流信息、行业资讯和手工 CSV。
2. 价格监控：识别黄瓜、番茄、鸡蛋、猪肉等品类的批发价、到货价、产地价、单位价、区域价差和异常波动。
3. 供应动态分析：识别新产季上市、新品种、新产区、新规格、新批次供应和到货量变化。
4. 质量与监管风险分析：识别食品安全、抽检不合格、农残/兽残、腐损率、缺斤少两、供应短缺、天气物流影响和监管通报。
5. 趋势对比：按日/周/月沉淀行情和风险数据，支持区域供应源之间的对标分析。
6. 报告生成：输出面向采购决策的区域供应源竞品动态追踪与智能对标分析报告。

系统不绕过登录、验证码、反爬，不抓取隐私或付费内容。无法稳定公开抓取的数据通过公开样例手工 CSV、授权 API 或规范化文件导入。
