### 1. LangChain 核心组件详解

#### 1.1 PromptTemplate

- **作用**：定义与 LLM 交互的提示模板，支持变量占位符。
- **示例**：
  
  ```python
  from dotenv import load_dotenv
  import os
  from langchain_openai import ChatOpenAI
  from langchain_core.prompts import PromptTemplate
  from langchain_core.output_parsers import StrOutputParser
  
  # 加载密钥
  load_dotenv(dotenv_path="../.env")
  
  # 初始化DeepSeek
  llm = ChatOpenAI(
      model=os.getenv("DEEPSEEK_MODEL", "deepseek-v4-flash"),
      openai_api_key=os.getenv("DEEPSEEK_API_KEY"),
      base_url="https://api.deepseek.com/v1",
      temperature=0.0,
  )
  
  # 定义提示词模板
  template = """
  你是一个生鲜批发采购区域供应源竞品情报分析师。请根据以下信息，生成一份简洁的竞品分析摘要。
  分析品类：{product}
  区域供应源竞品：{competitor}
  关键差异点：{key_points}
  """
  prompt = PromptTemplate(
      input_variables=["product", "competitor", "key_points"],
      template=template
  )
  
  # 构建LCEL链路
  chain = prompt | llm | StrOutputParser()
  
  # 传入参数调用并输出结果
  result = chain.invoke({
      "product": "黄瓜",
      "competitor": "河北黄瓜",
      "key_points": "山东寿光黄瓜批发价小幅上行，河北黄瓜受降雨影响到货减少，辽宁批发市场低价到货"
  })
  
  # 打印输出结果
  print("=== 竞品分析摘要 ===")
  print(result)
  ```

#### 1.2 LLMChain

- **作用**：将 PromptTemplate 与 LLM 客户端组合成一个调用链。
- **示例**（使用 DeepSeek 模型，通过 ChatOpenAI 调用）：

```
from langchain.chains import LLMChain
from langchain_openai import ChatOpenAI

# 配置 DeepSeek（兼容 OpenAI API）
llm = ChatOpenAI(
    model="deepseek-chat",
    openai_api_key="your-deepseek-api-key",
    openai_api_base="https://api.deepseek.com/v1",
    temperature=0.7
)

chain = LLMChain(llm=llm, prompt=prompt)
result = chain.run(product="黄瓜", competitor="河北黄瓜", key_points="批发价、到货量、质量风险")
print(result)
```

#### 1.3 RunnableSequence

- **作用**：LangChain Expression Language (LCEL) 中用于串联多个 Runnable 组件，形成复杂流水线。
- **示例**：先格式化输入，再调用 LLM，最后解析输出。
  
  ```python
  from dotenv import load_dotenv
  import os
  from langchain_openai import ChatOpenAI
  from langchain_core.prompts import PromptTemplate
  from langchain_core.runnables import RunnablePassthrough, RunnableLambda
  from langchain_core.output_parsers import StrOutputParser
  
  # 加载环境变量
  load_dotenv(dotenv_path="../.env")
  
  # 1. 初始化DeepSeek模型
  llm = ChatOpenAI(
      model="deepseek-chat",
      api_key=os.getenv("DEEPSEEK_API_KEY"),
      base_url=os.getenv("DEEPSEEK_BASE_URL"),
      temperature=0.4
  )
  
  # 2. 定义竞品分析Prompt模板
  template = """
  你是一个生鲜批发采购区域供应源竞品情报分析师。请根据以下信息，生成一份简洁的竞品分析摘要。
  分析品类：{product}
  区域供应源竞品：{competitor}
  关键差异点：{key_points}
  """
  prompt = PromptTemplate(
      input_variables=["product", "competitor", "key_points"],
      template=template
  )
  
  # 3. 基础串行RunnableSequence（透传完整入参，直接输出结果）
  sequence = (
      {"product": RunnablePassthrough(), "competitor": RunnablePassthrough(), "key_points": RunnablePassthrough()}
      | prompt
      | llm
      | StrOutputParser()
  )
  
  # 测试基础链路
  print("===== 基础RunnableSequence输出 =====")
  res1 = sequence.invoke({
      "product": "黄瓜",
      "competitor": "山东寿光黄瓜",
      "key_points": "规格稳定、批发价波动、区域价差"
  })
  print(res1)
  
  # 4. 自定义处理函数：截取文本前50字符作为精简关键词
  def extract_keywords(text):
      return text[:50]
  
  # 带预处理逻辑的复合链路
  sequence_with_parse = (
      {"raw_text": RunnablePassthrough()}
      | RunnableLambda(lambda x: extract_keywords(x["raw_text"]))
      | (lambda short_text: {
          "product": "黄瓜",
          "competitor": "河北黄瓜",
          "key_points": short_text
      })
      | prompt
      | llm
      | StrOutputParser()
  )
  
  # 测试带预处理的链路
  print("\n===== 带RunnableLambda预处理链路输出 =====")
  raw_info = "山东寿光黄瓜批发价小幅上行，河北黄瓜受降雨影响到货减少，辽宁批发市场低价到货"
  res2 = sequence_with_parse.invoke({"raw_text": raw_info})
  print(res2)
  ```

### 2. Skill 框架概述（通用设计）

- **SKILL.md 规范**：每个 Skill 是一个独立的目录，包含 `SKILL.md` 文件，用于描述 Skill 的名称、描述、触发条件、依赖、使用说明等。该规范不依赖特定 LLM 提供商，可适用于任何 Agent 或工具系统。

- **触发机制**：可通过关键词匹配、LLM 意图识别或显式调用触发。

- **描述语言设计**：采用 Markdown + YAML Frontmatter 结构，例如：

```
---
name: competitor-analysis
description: 分析指定竞品的情报信息，生成对比报告
trigger_keywords: ["竞品", "竞争对手", "分析"]
---
# 详细说明...
```

## 下午课程（理论 + 实操）

### 1. LangChain Memory 模块

- **作用**：管理多轮对话的上下文，支持多种内存类型（`ConversationBufferMemory`、`ConversationSummaryMemory`、`VectorStoreRetrieverMemory` 等）。
- **常用方法**：
  
  ```python
  from langchain.memory import ConversationBufferMemory
  from langchain.chains import ConversationChain
  
  memory = ConversationBufferMemory()
  conversation = ConversationChain(llm=llm, memory=memory)
  print(conversation.predict("你好，我想了解山东寿光黄瓜的批发价走势"))
  print(conversation.predict("那它和河北黄瓜相比呢？"))  # 自动记住上文
  ```

### 2. 配置 Python 依赖管理

- **pip 方式**：

    pip install pandas requests beautifulsoup4 langchain langchain-openai

- **poetry 方式**（推荐）：

     poetry init
     poetry add pandas requests beautifulsoup4 langchain langchain-openai



- **验证安装**：在 Python 交互环境中 `import` 各库，检查版本。

- **常见冲突排查**：
  
  - 版本不兼容（如 `langchain` 与 `langchain-openai` 版本不对应）
  
  - 依赖 `urllib3`/`requests` 冲突（可使用 `pip check` 或 `poetry check`）
  
  - 解决：使用虚拟环境（`venv` 或 `conda`），锁定版本 `==`。
